MIRAS

Desarrollo de la identidad visual para el Museo Interactivo Regional de Agroecología y Sustentabilidad ubicado en Villarica. Se desarrolló un manual de marca, el cual se usó de referencia para aplicarlo en una web con template.

Año

2021

Rol

Diseñadora gráfica y web

Herramientas

Illustrator, Photoshop, HTML/CSS

Contexto

MIRAS es un museo interactivo ubicado en las orillas del lago Villarica. Es un espacio de descubrimiento, participación y reunión en torno a una huerta agroecológica y todos los organismos que cohabitan en la cuenca. El equipo del museo contactó al Taller Lowtech de la Escuela de Diseño UC, para diseñar módulos interactivos en el museo y con ello, la identidad gráfica. El proceso de creación de marca estuvo inspirado en la cuenca del lago Villarica y como MIRAS se presenta como un espacio de reunión de diversos organismos en este espacio.

Objetivos

• Desarrollar la identidad visual de MIRAS, incluyendo la creación de un manuial de marca detallado abarcando logotipo, paleta de colores, tipografías, aplicación y estilo de ilustración/fotografías
‍
• Implementar la identidad visual creada en una propuesta de página web, partiendo de un template base para garantizar coherencia y consistencia de los elementos visuales.

Concepto

El logotipo de MIRAS busca representar lo orgánico, y un carácter ecosistémico donde todo está conectado y fluye. Nada es estático. La tipografía se eligió por las curvas, y como estas hacen referencia a la fluidez y cauce de un río. Además, se utilizó el concepto de "develar la fuerza escondida del agua", para determinar la paleta de colores, y el estilo de ilustraciones a utilizar.

Web

Se entregó una propuesta de web para MIRAS. El desarrollo de esta fue a partir de un template de themeforest, para luego modificarlo a través de HTML y CSS, con la imagen de la marca.
